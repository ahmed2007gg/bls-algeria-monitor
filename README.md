# 🔔 BLS Algeria Visa Appointment Monitor

بوت Python يراقب مواعيد تأشيرة إسبانيا على موقع BLS الجزائر ويرسل تنبيهات تلقرام بعدد المواعيد المتوفرة.

## ✨ المميزات

- مراقبة **4 مراكز**: الجزائر العاصمة، وهران، قسنطينة، عنابة
- تنبيهات تلقرام بالعربي مع عدد المواعيد المتوفرة لكل مركز
- تأخير عشوائي (25-40 ثانية) لتجنب الحظر
- يعيد تشغيل نفسه تلقائياً عند أي خطأ
- يعمل في الخلفية (Headless Chrome)

## 📦 التثبيت

```bash
pip install playwright requests
playwright install chromium
```

## 🚀 التشغيل

```bash
python bls_monitor.py
```

## ⚙️ الإعدادات

في أعلى ملف `bls_monitor.py`:

| المتغير | الوصف |
|---------|-------|
| `TELEGRAM_BOT_TOKEN` | توكن البوت من @BotFather |
| `TELEGRAM_CHAT_ID` | Chat ID الخاص بك |
| `HEADLESS` | True = بدون نافذة، False = مع نافذة |
| `MIN_DELAY / MAX_DELAY` | نطاق التأخير بالثواني |

## 🛑 إيقاف البوت

```
Ctrl+C
```
